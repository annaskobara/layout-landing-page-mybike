# MyBIKE Landing Page

A modern landing page inspired by the MyBIKE brand. This project showcases modern design layouts with a strong emphasis on precise grid alignment and fluid responsiveness across all devices.

## 📝 Description
This is a responsive landing page built as part of an educational project. It demonstrates the ability to translate high-fidelity designer mockups into clean, semantic and production-ready code while maintaining high standards of code organization and responsiveness.

## 🛠 Technologies
* **HTML5** — semantic and accessible markup structure.
* **SCSS** — modular styles organized using BEM methodology.
* **Gulp / Parcel** — asset bundling and development workflow automation.

## 🎯 Key Features
* **Pixel-Perfect Execution** — developed to match the original Figma design proportions, typography weights, and spacing.
* **Fully Responsive Layout** — fluid transitions across Mobile, Tablet, and Desktop layouts using mobile-first responsive design.
* **CSS Grid & Flexbox Mastered** — complex product presentation and structured layouts built for flexible content flow.
* **Clean and semantic HTML markup** using BEM methodology.
* **Smooth navigation** — fully functional and optimized responsive menu navigation.

## ▶️ How to run
* **Clone repository:** `git clone https://github.com/annaskobara/layout-landing-page-mybike.git`
* **Navigate into the project directory:** `cd layout-landing-page-mybike`
* **Install the dependencies:** `npm install`
* **Run the app:** `npm start`
